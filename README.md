# Teslo API 

1. Clonar el Proyecto
2. ```yarn install```
3. Clonar el Archivo ```.env.template``` y renonmbrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levantar la DB
```
docker-compose up -d 
```
6. Levantar en modo de Dev:
```
yarn start:dev
```
